  # Classnote
  **Fully-featured school-note management application.**
</div>

## Overview
**Classnote** allows you to effortlessly manage your school work and notes, schedule office hours, remember questions for later, and chronologically organize coursework.

---

## Quick Start
0. **Prerequisites:** You must have [`git`](https://git-scm.com/install/) and [`cargo`](https://doc.rust-lang.org/cargo/getting-started/installation.html) installed.
1. **Install:** Clone the source code (binary releases coming soon), and run `cargo i` to build and install the binary.
2. **Initialize:** Run `classnote` with no arguments to create and initialize the Classnote configuration file.
3. **Start:** After the configuration is complete, Classnote will automatically open the note for the class corresponding to current time, if one exists.
4. **Manage:** Management options coming soon!

For help, run `classnote help`.
